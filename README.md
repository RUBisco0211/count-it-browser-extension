# Count It 字数统计

轻量的**中文字数统计**浏览器扩展。可用于**Chromium**内核浏览器

## 功能

选中文本后右键菜单**字数统计**或直接点击扩展图标以查看当前选区的字数统计。

![](./assets/screenshot.png)

## 统计项

- 字符数（含空白）
- 字符数（不含空白）
- 中文字符数
- 非中文单词数
- 英文字符数
- 数字数
- 标点数

## 权限说明

- `contextMenus`：添加右键菜单。
- `tabs` / `activeTab` / `scripting`：读取当前页面选区（弹窗打开时）。
